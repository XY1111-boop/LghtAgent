### 📚 知识卡片：爬虫 requests 库

**标题**：Python requests 库基础用法与爬虫规范

**关键概念**：
- **requests.get()**：发送 HTTP GET 请求的核心方法。
- **params 参数**：用于在 URL 中添加查询字符串（如 `?key=value`）。
- **响应对象属性**：
  - `resp.status_code`：获取 HTTP 状态码（如 200 表示成功）。
  - `resp.json()`：将响应内容解析为 JSON 格式（适用于 API 返回）。
- **headers 模拟**：通过添加请求头（如 `User-Agent`）伪装成浏览器，避免被服务器识别为爬虫。
- **爬虫规范**：
  - 遵守 `robots.txt`：尊重网站的爬取规则。
  - 不要超频：控制请求频率，避免对服务器造成压力。
  - 尊重版权：不滥用爬取数据，遵守法律法规。

**摘要**：
本文档介绍了 Python `requests` 库的基本用法，包括如何发送 GET 请求并携带参数、获取状态码和 JSON 数据，以及通过添加 headers 模拟浏览器行为。同时强调了爬虫开发中必须遵守的三项重要规范：遵守 `robots.txt`、控制请求频率、尊重数据版权，以确保爬虫的合法性与道德性。
