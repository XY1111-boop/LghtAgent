### 知识卡片：SQLite 数据库基本操作与安全查询

**标题**：SQLite 数据库操作与 SQL 注入防范

**关键概念**：
- **数据库连接**：使用 `sqlite3.connect('test.db')` 创建或连接一个 SQLite 数据库文件。
- **游标对象**：通过 `conn.cursor()` 获取游标，用于执行 SQL 语句。
- **创建表**：使用 `c.execute("CREATE TABLE ...")` 执行建表语句。
- **SQL 注入防范**：避免直接拼接 SQL 字符串，应使用**参数化查询**（如 `?` 占位符）来安全传递用户输入。

**摘要**：
本文档演示了使用 Python 的 `sqlite3` 模块进行数据库操作的基本流程，包括连接数据库、获取游标以及创建表。同时重点强调了 SQL 注入的防范措施，推荐使用参数化查询（例如 `c.execute("INSERT INTO users VALUES (?, ?)", (id, name))`）来替代字符串拼接，从而提升数据库操作的安全性。
