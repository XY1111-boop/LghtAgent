### 📚 知识卡片：Git 基础

**标题**：Git 基础操作与协同开发

**关键概念**：
- **git init / clone**：初始化本地仓库 / 克隆远程仓库
- **git add / commit**：暂存更改 / 提交到本地仓库
- **git push / pull**：推送本地提交到远程 / 拉取远程更新到本地
- **git branch / merge**：创建分支 / 合并分支
- **.gitignore**：忽略不需要版本控制的文件（如日志、编译产物等）
- **协同开发**：通过分支管理、远程同步实现多人协作

**摘要**：
Git 是目前最流行的分布式版本控制系统。通过 `init`、`clone`、`add`、`commit`、`push`、`pull`、`branch`、`merge` 等核心命令，开发者可以高效管理代码版本、避免代码丢失。配合 `.gitignore` 文件忽略不必要的文件，能有效提升团队协同开发的效率与安全性。掌握 Git 是现代软件开发的基础技能之一。
