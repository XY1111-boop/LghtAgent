### 📘 知识卡片：23种设计模式（Python 版）

**标题**：23种设计模式 Python 版

**关键概念**：
- **设计模式**：在软件工程中，针对常见问题的可复用解决方案。
- **单例模式**：确保一个类只有一个实例，并提供一个全局访问点。
- **工厂模式**：定义一个创建对象的接口，让子类决定实例化哪个类。
- **观察者模式**：定义对象间的一对多依赖，当一个对象状态变化时，所有依赖者自动收到通知。
- **策略模式**：定义一系列算法，将每个算法封装起来，并使它们可以互相替换。
- **装饰器模式**：动态地给对象添加额外的职责，比继承更灵活。
- **Python 实现**：利用 `__new__`、`__init__`、装饰器、类方法等 Python 特性实现设计模式。

**摘要**：
本文档介绍了 23 种经典设计模式在 Python 中的实现方式，并以单例模式为例展示了如何通过 `__new__` 方法控制实例创建。设计模式能够提升代码的可维护性、复用性和扩展性，但应避免过度设计，仅在确实需要时使用。常见模式包括单例、工厂、观察者、策略、装饰器等，每种模式都有其适用场景和 Python 特有的实现技巧。
