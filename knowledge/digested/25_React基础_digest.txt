**标题**：React 核心概念：组件、状态与 Props

**关键概念**：
- **组件**：React 应用的基本构建块，用于封装 UI 和逻辑。
- **状态 (State)**：组件内部管理的数据，通过 `useState` 定义，状态变化会触发组件重新渲染。
- **Props**：父组件传递给子组件的只读属性，用于组件间通信。
- **useState**：React Hook，用于在函数组件中添加和管理状态。
- **useEffect**：React Hook，用于处理副作用（如数据获取、订阅、DOM 操作）。
- **JSX 语法**：JavaScript 的语法扩展，允许在 JavaScript 中编写类似 HTML 的标记。

**摘要**：
本文档介绍了 React 框架中的三个核心概念：组件、状态（State）和属性（Props）。组件是构建用户界面的独立单元；状态通过 `useState` 钩子管理，用于存储和更新组件内部数据；Props 则用于父组件向子组件传递数据。此外，文档还提到了 `useEffect` 钩子，用于处理组件中的副作用操作，以及 JSX 语法，它允许开发者以声明式的方式在 JavaScript 中编写 UI 结构。
